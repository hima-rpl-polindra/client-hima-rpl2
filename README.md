Client-HIMA RPL

HIMPUNAN MAHASISWA PERANGKAT LUNAK - POLITEKNIK NEGERI INDRAMAYU

---

## 🚀 Main Feature

- Blog
- Posting
- Informasi

---

## 🛠️ Tech Stack

- Frontend: React.js / Next.js
- Backend: Express.js / Node.js
- Database: MongoDB
- Storage: Cloudinary
- Lainnya: Tailwind CSS

---

## 🧑‍💻 Install (local)

```bash
# Clone Repository
git clone https://github.com/hima-rpl-polindra/client-hima-rpl2.git

# Masuk ke folder project
cd client-hima-rpl2

# Install Dependency
npm install

# Jalankan Aplikasi
npm run dev

# Jalankan Tailwind
npm run tailwind

```
